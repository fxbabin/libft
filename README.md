# Libft
            
The libft project consists in recoding some functions of the C standard library. The project will be used for other projects as a base of useful functions.

# Available functions

## Memomy functions 
* ft_memset
* ft_bzero
* ft_memalloc
* ft_memcpy
* ft_memccpy
* ft_memmove
* ft_memchr
* ft_memchrindex
* ft_memcmp
* ft_memdel
* ft_int2alloc
* ft_char2alloc
* ft_memalloc2
* ft_free2
* ft_memdup
